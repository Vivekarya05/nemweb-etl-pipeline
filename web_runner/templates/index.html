AEMO NEMWeb Market Data Pipeline

Download, clean, validate, and load Australian electricity market data into PostgreSQL.

Select a date range, choose an analysis model, review the recommended datasets, and run the pipeline. The app checks for existing data first, avoids duplicate loads, and stores clean output for pgAdmin, Power BI, and Python workflows.
